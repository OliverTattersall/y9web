Explains and Justifies

Child Mortality is a huge issue around the world but many people don't know the high numbers of child deaths and information is all in boring tabular information which is hard to digest. Giving a easier, more digestable form of the data will further educate people and encourage them to help.

Constructs a detailed research plan

  1. Learn about charts on the canvas tag to allow myself to create a better visual representation of the data.
  2. Learn about the dropdown bar tag, option, to give the user a way to select which countries it wants to look at.
  3. Learn more about array manipulation so that I can better use the data.
